Muchos Sistemas Operativos

Además de Linux hay otros Sistemas Operativos/Núcleos

  • OpenBSD : Seguridad.

  • Plan 9 : Sistema Distribuido.

  • NetBSD : Más de 50 arquitecturas.

  • Be/OS : El escritorio.

  • OpenMosix : Modificación de Linux para ambiente Distribuido

El sistema operativo es el conjunto de componentes que hacen del computador algo funcional, el núcleo es el encargado de coordinar procesos básicos, para que haya una infraestructura sobre la cuál construir más.
